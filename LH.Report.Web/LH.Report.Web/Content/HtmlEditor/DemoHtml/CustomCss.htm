What's New in DXperience v2010 vol 2

ASP.NET HTML Editor Control

Image Selector

  • We have enhanced the Insert Image and Change Image dialogs with the ability to invoke a Select Image dialog - allowing end-users to manage content images via a point-and-click UI, rather than typing image web addresses manually. Because our new File Manager control is integrated into the Select Image dialog, all file/folder management and upload capabilities are available to you out-of-the-box... with full customization.


Figure 1. Select Image Dialog

Content Validation

  • This release introduces a comprehensive API enabling you to validate editor content, both on the client and server. Among the newly introduced members are validation events and various settings for a built-in frame that displays validation errors.
